Billetera Neexa

Billetera Neexa - Aplicación Financiera Inteligente

Billetera Neexa es una aplicación web completa que simula una billetera digital moderna con funcionalidades avanzadas de gestión financiera. Desarrollada con React + TypeScript en el frontend y Python + Flask en el backend, ofrece una experiencia completa de banca digital que incluye gestión de ahorros, análisis financiero y conversión de monedas en tiempo real.

Características Principales

  • • Dashboard interactivo con visualización de saldo y tarjeta virtual
  • • Sistema de ahorros inteligente con metas personalizables y retos gamificados
  • • Conversor de monedas con tasas de cambio en tiempo real
  • • Análisis financiero con recomendaciones basadas en patrones de gasto
  • • Autenticación segura con JWT y encriptación de contraseñas
  • • Modo oscuro/claro con diseño responsivo

Funcionalidades Destacadas

  • • Gestión de presupuestos por categorías con análisis inteligente
  • • Conversor de divisas integrado con tasas actualizadas
  • • Sistema de recomendaciones simulada para optimización
  • • Transferencias y depósitos virtuales con historial completo
  • • Análisis de patrones de gasto y proyecciones financieras
  • • Interfaz intuitiva con componentes reutilizables

Neexi Asistente Virtual

Sobre el Proyecto

Neexi es un asistente virtual especializado en educación financiera que transforma conceptos complejos de ahorro y presupuesto en conversaciones amigables y desafíos divertidos. Diseñado para hacer que la gestión financiera personal sea accesible para todos.

El asistente guía a los usuarios a través de diferentes flujos conversacionales, desde la creación de presupuestos personalizados hasta la participación en retos de ahorro que hacen que el proceso sea entretenido y motivador.

Características Principales

  • • Conversaciones naturales sobre finanzas personales
  • • Creación de presupuestos personalizados
  • • Retos de ahorro semanal y diario
  • • Consejos financieros adaptados al usuario
  • • Interfaz conversacional intuitiva
  • • Cálculos automáticos de ahorro

Tecnologías Utilizadas

Plataforma

Voiceflow - Diseño visual de flujos conversacionales

Funcionalidades

IA Conversacional, Lógica de Flujo, Variables Dinámicas

Cómo se Desarrolló

El proyecto se desarrolló utilizando Voiceflow, una plataforma visual que permite crear asistentes conversacionales sin necesidad de programación tradicional. Se diseñaron flujos de conversación intuitivos que guían al usuario a través de diferentes opciones, desde la creación de presupuestos hasta la participación en retos de ahorro. La lógica se implementó mediante bloques visuales que manejan variables, cálculos automáticos y respuestas personalizadas, creando una experiencia fluida y educativa.
